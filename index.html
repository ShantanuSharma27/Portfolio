Baatein

Baatein is a dynamic web application designed for seamless real-time communication with an engaging chatting interface.It features a real-time typing indicator for instant feedback, enhancing user interaction. Media files are efficiently managed through Cloudinary, and real-time message translation is powered by Socket.io, allowing for barrier-free communication across languages. Developed using React JS for the front-end, Express JS and Node JS for the back-end, and MongoDB for database management, the application is styled with Chakra UI for a modern, responsive design.

02

Study Notion

Study Notion is an Edtech platform designed to enhance education in rural areas. It supports three roles: instructorswho upload courses, students who access them, and administrators who manage the system. The platform uses Cloudinary for media storage, JWT for secure authorization, and NodeMailer for OTP-based authentication. Tailwind provides styling, while React and Redux Toolkit create the frontend, Node.js handles the backend, and MongoDB serves as the database.Razorpay is integrated for smooth payment transactions.
